# Object-Oriented Programming (OOP) in Java

This repository covers all fundamental concepts of Object-Oriented Programming (OOP) using Java. It is designed for beginners and students who want to build a strong understanding of core OOP principles.

## 📚 Topics Covered

* Classes & Objects
* Access Modifiers
* Getters & Setters
* Encapsulation
* Constructors & Types
* Copy Constructor
* Shallow & Deep Copy
* Destructors
* Inheritance (Single, Multilevel, Hierarchical, Hybrid)
* Polymorphism
* Method Overloading & Overriding
* Packages in Java
* Abstraction
* Abstract Classes
* Interfaces
* Static & Super Keywords

## 🎯 Purpose

* Strengthen OOP fundamentals
* Help in academic preparation
* Provide quick revision notes
* Practice coding problems

## 🧑‍💻 Who Can Use This?

* Beginners learning Java
* Engineering students
* Anyone preparing for coding interviews

## 🚀 How to Use

Go through the concepts step by step and try implementing the examples yourself. Practice questions are included to test your understanding.

## 📌 Note

This repository focuses on clarity and simplicity, making it easy to grasp even complex OOP concepts.
